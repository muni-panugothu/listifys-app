# Listify App Repository

This repository is now organized so the mobile Expo app lives inside the `app/` folder.

## Structure

- `app/` - Expo React Native application
- `server/` - reserved for a future backend service

## Mobile App

Run app commands from the mobile app folder:

```bash
cd app
npm install
npm run start
```

Common commands:

```bash
cd app
npm run android
npm run lint
```
