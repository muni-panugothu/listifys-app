# Listifys Design System & Part 1 Requirements

## Brand Tokens
- **Primary:** #27BB97
- **Primary Dark:** #1FA987
- **Gradient:** #27BB97 to #1E9E7E
- **Secondary Blue:** #2D7DD7
- **Accent Yellow:** #FFCE32
- **Background:** #FFFFFF
- **Card Border:** #F3F4F6
- **Heading:** #111827
- **Body:** #4B5563
- **Muted:** #9CA3AF
- **Error:** #EF4444 (Background: #FEF2F2)
- **Success:** #10B981
- **Warning:** #F59E0B

## UI Geometry & Effects
- **Radius:** 
  - Cards/Modals: 12px
  - Inputs/Buttons: 8px
  - Chips/Avatars: Fully Rounded
- **Shadows:**
  - Rest: 0 1px 2px rgba(0,0,0,0.05)
  - Hover/Tap: 0 4px 12px rgba(0,0,0,0.1)
  - Modal/Sheet: 0 8px 30px rgba(0,0,0,0.12)
- **Typography:** Inter (400 to 700)
- **Icon Style:** Lucide-like thin stroke

## Global App Shell
- **Bottom Tabs:** Home, Search, Sell (Center FAB), Messages, Profile
- **Top App Bar (Marketplace):** Logo (Left), Location Selector (Center), Notifications (Right)
- **Listing Cards:** Image, Badge, Glassmorphism Heart, 2-line title, INR price (₹), Location
